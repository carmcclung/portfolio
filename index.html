Content Developer and Technical Writer

An accomplished technical writer with diverse experience writing user-facing content for software and medical devices. Significant experience with single-source authoring and DITA XML, as well as publishing in a Docs as Code setting.

Career Highlights

    • This project involved converting 41 total pieces of documentation from Microsoft Word to DITA XML. The content included 26 instructions for use (IFUs) and two manuals and was migrated into a CMS to facilitate significant content reuse across the products. During this process I supervised a contract writer who was hired to support the project, I collaborated with multiple stakeholders and SMEs across the organization, and I led the effort to incorporate the product facing documentation into the companies' processes.
  • Created a process for managing UI text development for an enterprise software project that spanned eleven distributed teams.
  • Oversaw and produced the labeling and user documentation for several successful regulatory submission processes, such as 510(k).
  • Developed suites of user facing documentation for state-of-the-art medical devices and a cloud-based software application.
  • Contributed significantly to solutions for content reuse and translation issues in a CMS.
  • Wrote, edited, and maintained thousands of lines of text that appear on user interfaces for both devices with embedded software and standalone software applications.
